Початок роботи

Ласкаво просимо до Медіа-каталогу! Цей посібник допоможе вам швидко освоїти основні функції системи.

Основні кроки:

1
Реєстрація та вхід
Створіть обліковий запис або увійдіть у систему для доступу до вашої персональної колекції.
2
Ознайомлення з інтерфейсом
Дослідіть головну панель, каталог та інші розділи для кращого розуміння структури.
3
Налаштування профілю
Налаштуйте тему, мову та інші параметри відповідно до ваших уподобань.

Додавання медіа

Додавайте фільми, книги та інші медіа до вашої колекції кількома способами:

Ручне додавання

1
Відкрийте форму додавання
Натисніть кнопку "Додати медіа" у каталозі.
2
Оберіть тип медіа
Виберіть "Фільм" або "Книга" залежно від типу контенту.
3
Заповніть інформацію
Введіть назву, рік, рейтинг, опис та інші деталі.

Автоматичне додавання через API

Система автоматично заповнює інформацію при пошуку через інтегровані API:

// Автоматичне отримання даних з TMDB та Google Books
function searchMedia(query) {
  return Promise.all([
    searchTMDB(query),
    searchGoogleBooks(query)
  ]);
}

Створення книг

Додавання книг до вашої медіа-колекції має особливі можливості:

Кроки створення книги:

1
Оберіть тип "Книга"
У формі додавання медіа оберіть опцію "Книга".
2
Заповніть основну інформацію
  • Назва книги - повна назва твору
  • Автор - ім'я автора або авторів
  • Рік видання - рік публікації
  • Жанр - літературний жанр
3
Додайте додаткові деталі
  • ISBN - унікальний ідентифікатор книги
  • Кількість сторінок - обсяг видання
  • Опис - анотація або короткий зміст
  • Обкладинка - URL зображення обкладинки
4
Використовуйте автоматичне заповнення
Система автоматично заповнить інформацію при введенні ISBN або назви книги через інтеграцію з Google Books API.

Особливості роботи з книгами:

ISBN пошук
Автоматичне отримання інформації по ISBN коду
Кілька авторів
Підтримка декількох авторів для однієї книги
Категорії
Організація книг за жанрами та категоріями

Пошук та фільтри

Ефективний пошук та фільтрація допоможуть швидко знайти потрібне медіа:

Типи пошуку:

  • Текстовий пошук - пошук за назвою, описом, автором тощо
  • Фільтрація за типом - фільми, книги, улюблені
  • Фільтрація за рейтингом - пошук медіа з високим рейтингом
  • Фільтрація за роком - пошук за роком випуску

Розширені можливості:

1
Комбіновані фільтри
Поєднуйте різні типи фільтрів для точного пошуку.
2
Збереження пошуків
Зберігайте улюблені пошукові запити для швидкого доступу.
3
Історія пошуку
Переглядайте та повторюйте останні пошукові запити.

Голосовий пошук

Використовуйте голосовий пошук для швидкого та зручного пошуку медіа:

Як користуватись:

1
Натисніть мікрофон
Клацніть на іконку мікрофона у полі пошуку.
2
Говоріть запит
Чітко проговоріть назву фільму, книги або автора.
3
Отримайте результати
Система автоматично виконає пошук за вашим голосовим запитом.

Поради для кращого розпізнавання:

  • Говоріть чітко та не надто швидко
  • Використовуйте повні назви для точнішого пошуку
  • Переконайтесь, що у вас є доступ до мікрофона
  • Використовуйте функцію в тихому середовищі

Вирішення проблем

Якщо ви зіткнулися з проблемами, ось деякі рішення:

Поширені проблеми та рішення:

1
Медіа не додається
Перевірте підключення до інтернету та спробуйте ще раз. Переконайтесь, що всі обов'язкові поля заповнені.
2
Пошук не працює
Очистіть кеш браузера та перезавантажте сторінку. Перевірте, чи не заблоковані API запити.
3
Голосовий пошук не активний
Дозвольте доступ до мікрофона у налаштуваннях браузера. Переконайтесь, що використовуєте HTTPS з'єднання.

Як повідомити про проблему:

Якщо проблема не вирішена, будь ласка:

  1. Зробіть скріншот проблеми
  2. Запишіть кроки для відтворення проблеми
  3. Перейдіть до розділу Контакти
  4. Надішліть детальний опис проблеми

Часті запитання

Швидкі відповіді на найпопулярніші питання

Як додати книгу без ISBN?

Ви можете додати книгу без ISBN, заповнивши інформацію вручну. Просто введіть назву, автора, рік видання та інші деталі у відповідних полях форми додавання.

Чи можна імпортувати дані з інших сервісів?

Так, система підтримує імпорт даних з популярних сервісів через API. Для детальної інформації зверніться до розділу "Додавання медіа".

Як працює голосовий пошук?

Голосовий пошук використовує Web Speech API. Натисніть на іконку мікрофона, дозвольте доступ та чітко проговоріть пошуковий запит. Система автоматично розпізнає та виконає пошук.

Чи зберігаються мої дані при оновленні сторінки?

Так, всі ваші дані зберігаються локально у браузері та на сервері. Ваша колекція залишиться недоторканою навіть після оновлення сторінки чи перезавантаження браузера.
